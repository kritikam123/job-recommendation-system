# Job Recommendation System

A content-based job recommendation system that suggests similar jobs based on job title using cosine similarity.

## Project Overview
This project recommends jobs by analyzing similarities between job features such as job title, skills, and experience. It is designed as a real-world data science application and deployed using Streamlit.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Pickle

##  Recommendation Technique
- Content-Based Filtering
- Cosine Similarity

## Features
- Select a job title
- Get top 5 similar job recommendations
- Interactive web interface using Streamlit
